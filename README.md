# Computer-lab-works# 2nd Semester: C++ Programming Lab Works 🚀

Welcome to my repository for the **2nd Semester C++ Programming** laboratory assignments. This repository contains all my foundational C++ source code, problem sheets, and practical implementations completed during this semester.

---

## 📚 Core Topics Covered

This lab series covers the fundamentals of C++ and introduces Object-Oriented Programming (OOP) concepts:

* **Basics & Control Flow:** Input/Output (`cin`/`cout`), loops, functions, and arrays.
* **Structures & Pointers:** Memory management, basic structures, and pointer manipulation.
* **Object-Oriented Programming (OOP):**
  * Classes and Objects
  * Constructors and Destructors
  * Encapsulation and Data Hiding
  * Inheritance (Single, Multiple, Hierarchical)
  * Polymorphism (Function overloading and Virtual functions)
